  • Advent of Code

    • JavaScript
    • TypeScript
    • Algorithms & Data Structures

    Advent of Code is a yearly programming contest in December, made by Eric Wastl and hosted on adventofcode.com. The contest consist of 25 daily programming puzzles. Each puzzle consists of two parts: the first part will introduce the problem and is gives a hint to how the problem can be tackled. The second part will either continue building on the first part, or you will meet a barrier in you approach that does not cover the changed problem.

    I have competed in both 2020 and 2021. In 2020 my programming language of choice was JavaScript, for 2021 I switched to TypeScript with a custom environment, using a forked CLI to get personal input from the website.

    Advent of Code has helped me practice implementing different data structures and algorithms. It helped improve my coding fundamentals and to think outside of the box. An acquaintance did code reviews on the more difficult problems to help me point out areas to improve.

  • Korfball stats

    • Java
    • Android Studio

    With a new coach at the korfball club came a more analytical approach to the matches. Recording matches and logging statistics about scoring rates. In my opinion this was a big improvement, using data easily available to influence your decisions. But tallying scoring attempts and goals on a piece of paper, that wasn’t my thing.

    Using Adobe XD I made a wireframe and I wrote down some specifications about the application. In my spare time I build out the app in Java using Android Studio and sideloaded the application on my phone. A couple matches later, I was able to use the application to record the data. The big advantage was that the app would output a CSV file that could be used to visualize the data, no more counting lines or furiously deciphering handwriting.
